🚀 Quick Start

This page demonstrates the VisualFX debug panel functionality. The debug panel can be accessed from any page on the site using keyboard shortcuts.

Open/Close Debug Panel Ctrl + Shift + V
Close Panel (when open) ESC
Clear All Effects (when panel open) Ctrl + Shift + C
