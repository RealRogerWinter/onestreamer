# Netscape HTTP Cookie File
# https://curl.se/docs/http-cookies.html
#
# INSTRUCTIONS:
# YouTube is blocking some videos with bot detection.
# To fix this, you need to export your YouTube cookies from a logged-in browser.
#
# Steps:
# 1. Install a browser extension like "Get cookies.txt LOCALLY" for Chrome/Firefox
# 2. Go to youtube.com and make sure you're logged in
# 3. Click the extension icon and export cookies for youtube.com
# 4. Replace the contents of this file with the exported cookies
#
# The format should be:
# .youtube.com	TRUE	/	FALSE	<expiry>	<cookie_name>	<cookie_value>
#
# Example cookies you need:
# .youtube.com	TRUE	/	TRUE	<expiry>	__Secure-1PSID	<value>
# .youtube.com	TRUE	/	TRUE	<expiry>	__Secure-1PAPISID	<value>
# .youtube.com	TRUE	/	TRUE	<expiry>	__Secure-3PSID	<value>
#
# After adding cookies, restart the server: pm2 restart onestreamer-server
